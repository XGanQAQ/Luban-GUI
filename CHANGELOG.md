## [0.1.0] — 2026-04-15

### 早期预览版本（Preview）

> 说明：本版本为可用的早期预览版，以下内容按“当前已验证”与“已有入口待完善”区分。

#### 项目管理

- 新建项目：选择工作区根目录 + 填写项目名，自动生成完整 Luban 工作区（`luban.conf`、`Datas/`、三个元数据 xlsx）
- 打开已有 Luban 项目目录
- 多项目快速切换（顶部下拉栏），应用重启后自动恢复上次打开的项目
- 项目配置持久化到 `projectConfig.json`；全局注册表存储于 `%LOCALAPPDATA%\LubanGui\appConfig.json`

#### Schema 管理（快捷操作）

- **新建表格**：对话框填写表名与字段列表，自动生成符合 Luban 规范的 `.xlsx`（含 `##var`/`##type`/`##` 元数据行）并注册到 `__tables__.xlsx`
- **新建枚举**：已提供 UI 入口与创建流程，仍需补充完整场景验证
- **新建 Bean**：已提供 UI 入口与创建流程，仍需补充完整场景验证
- **导入文件**：将已有 xlsx 注册到对应元数据表

#### 表格预览

- 左侧表格列表面板：显示当前项目全部表格，支持实时名称过滤
- 右键菜单：在资源管理器中显示、从项目移除等操作
- 右侧内容面板：DataGrid 渲染表格数据行（跳过 `##` 元数据行），虚拟化渲染支持大数据量流畅滚动
- 双击列标题：调用系统默认 Excel 应用打开对应 `.xlsx` 文件

#### 导出配置

- 导出配置窗口（`Ctrl+,`）：数据格式下拉 + 数据输出路径
- 可选代码导出：语言类型 + 输出路径 + 命名空间
- 从 `luban.conf` 动态读取导出目标列表
- 配置自动保存，后续一键导表无需重填

#### 一键导表

- `F5` 触发全量导表，启动内嵌 Luban CLI 子进程
- 异步并行捕获 stdout / stderr，实时推送日志到 UI（无 UI 阻塞）
- 支持取消导表（菜单“操作 → 取消”）
- 导表完成后状态栏显示耗时与结果

#### 日志窗口

- `Ctrl+L` 打开独立浮动日志窗口
- 按日志级别着色（INFO / WARN / ERROR / SUCCESS）
- 自动滚动至最新条目，支持手动暂停自动滚动
- 支持 10,000+ 条日志流畅显示
- 清空日志按钮

#### 基础设施

- 五层架构：表现层 → 业务逻辑层 → 工具层 → 基础设施层 → Luban 源适配层
- 依赖注入（Microsoft.Extensions.DependencyInjection）贯穿全局
- Serilog 结构化日志（输出至控制台 + 滚动日志文件）

## [Unreleased]

### 新增（v0.2 进行中）

#### 已实现（v0.2 已落地）

##### Schema 与类型能力

- **新建枚举 / 新建 Bean**：已完成主流程，可创建并写入 `__enums__.xlsx` / `__beans__.xlsx`
- 新建自定义类型后，字段类型选择可即时感知并用于导表
- 容器类型支持已接入（覆盖常用 `list` / `set` / `map` 表达）

##### 预览与定位能力

- 表格预览右键菜单支持「定位到 Excel 单元格」
- 可从预览单元格映射并打开对应 xlsx 的目标行列位置

##### 删除表格（Week 4）

- 表格列表右键菜单新增「🗑 删除表格...」入口
- 二次确认对话框支持删除策略选择：
  - **仅移除注册**：从 `__tables__.xlsx` 中移除条目，保留 xlsx 数据文件
  - **同时删除物理文件**：额外删除对应的 xlsx 数据文件（不可恢复）
- 支持将当前策略设为默认（保存到 `appConfig.json`）
- 删除当前选中表时自动清空右侧预览区
- 所有操作记录结构化日志

##### 核心链路回归

- 典型项目下，删除后列表 / 预览 / 状态栏状态一致
- `__tables__.xlsx` 注册项可正确更新
- 全量导表主链路可通过（无阻断级缺陷）

#### 持续完善（v0.2 收尾）

- 重名与非法命名拦截提示仍需补齐更多场景验证
- 非法容器表达式字段级拦截提示仍需补齐
- 数据类型总览中的来源分类展示（内置 / 自定义）仍在完善
