# Luban-GUI 层次划分

> 技术选型、架构优势、NuGet 清单等内容见 [01-技术方案.md](./01-技术方案.md)。

---

## 五层架构概览

| 层次 | 文件夹 | 职责 | 主要组件 |
|------|--------|------|----------|
| **表现层** | [presentation-layer/](./presentation-layer/) | UI 显示与用户交互，路由命令到 ViewModel | 主窗口、导出配置窗口、日志窗口，5 个 ViewModel |
| **业务逻辑层** | [service-layer/](./service-layer/) | 实现核心业务流程，协调各 Service | ProjectManager、SchemaService、ExportService、TablePreviewService |
| **工具层** | [tool-layer/](./tool-layer/) | 封装底层 IO 和进程操作 | LubanExecutor、LubanCommandBuilder、ExcelWriter、FileOpenService |
| **基础设施层** | [infrastructure-layer/](./infrastructure-layer/) | 持久化配置，提供日志等横切关注点 | AppConfigManager、ProjectConfigManager、Logger |
| **Luban 源适配层** | [luban-adapter-layer/](./luban-adapter-layer/) | 隔离 GUI 与 Luban 源码 | LubanSchemaReader、LubanConfAdapter、LubanTypeMapper |

层间只向下依赖；上层通过接口调用下层，不直接访问下下层。**Luban 源适配层是 GUI 与 `lubanSrc\` 之间的唯一接触点**，上方各层不得直接引用 Luban 源码中的类型。

---

## 表现层 (Avalonia UI)

**文件夹**: [presentation-layer/](./presentation-layer/)

**职责**:
- 显示项目列表，支持切换
- 显示表格列表，支持快捷操作（新建 / 导入）
- 预览表格数据，支持双击列标题定位到 Excel 文件
- 提供导出配置窗口
- 路由命令到 ViewModel，不包含业务逻辑

**模块列表**:

| 文件 | 组件 | 职责 |
|------|------|------|
| [MainWindowViewModel.md](./presentation-layer/MainWindowViewModel.md) | `MainWindowViewModel` | 统筹项目切换、表格列表、表格预览、导表状态 |
| [TableListViewModel.md](./presentation-layer/TableListViewModel.md) | `TableListViewModel` | 管理左侧表格列表，响应快捷操作命令 |
| [TablePreviewViewModel.md](./presentation-layer/TablePreviewViewModel.md) | `TablePreviewViewModel` | 管理右侧表格数据预览，响应双击定位命令 |
| [ExportConfigViewModel.md](./presentation-layer/ExportConfigViewModel.md) | `ExportConfigViewModel` | 管理导出配置（数据类型 / 路径、代码类型 / 路径） |
| [LogWindowViewModel.md](./presentation-layer/LogWindowViewModel.md) | `LogWindowViewModel` | 管理日志浮动窗口 |

**层间约定**: 只调用业务逻辑层接口，通过构造函数注入 Service 接口，所有 UI 更新在 Avalonia UI 线程执行。

---

## 业务逻辑层 (Service)

**文件夹**: [service-layer/](./service-layer/)

**职责**: 实现核心业务流程，协调工具层完成复杂操作，不包含 UI 逻辑。

**模块列表**:

| 文件 | 组件 | 职责 |
|------|------|------|
| [ProjectManager.md](./service-layer/ProjectManager.md) | `ProjectManager` | 管理多项目的创建、切换、加载，初始化工作区 |
| [SchemaService.md](./service-layer/SchemaService.md) | `SchemaService` | 维护 Luban 元数据表，提供新建表 / 枚举 / Bean 操作 |
| [ExportService.md](./service-layer/ExportService.md) | `ExportService` | 根据 `ProjectConfig` 编排完整的导表流程 |
| [TablePreviewService.md](./service-layer/TablePreviewService.md) | `TablePreviewService` | 读取 Excel 数据文件，为右侧面板提供预览数据 |

**层间约定**: 向下依赖工具层和基础设施层接口，只暴露 `I*` 接口给表现层，所有 `Task` 方法保证线程安全。

---

## 工具层 (Executor / IO)

**文件夹**: [tool-layer/](./tool-layer/)

**职责**: 封装底层 IO 和进程操作，不包含业务逻辑，不引用任何 Service 接口。

**模块列表**:

| 文件 | 组件 | 职责 |
|------|------|------|
| [LubanExecutor.md](./tool-layer/LubanExecutor.md) | `LubanExecutor` | 启动 Luban CLI 子进程，捕获 stdout / stderr 输出流 |
| [LubanCommandBuilder.md](./tool-layer/LubanCommandBuilder.md) | `LubanCommandBuilder` | 将 `ProjectConfig` 转换为完整的 Luban CLI 参数 |
| [ExcelWriter.md](./tool-layer/ExcelWriter.md) | `ExcelWriter` | 按 Luban 规范读写 `.xlsx` 文件（使用 ClosedXML） |
| [FileOpenService.md](./tool-layer/FileOpenService.md) | `FileOpenService` | 打开文件并可选地定位到指定 Excel 单元格 |

**层间约定**: 纯 I/O / 进程封装，不依赖 Service 接口或 ViewModel，只接受基础数据类型作为参数。

---

## 基础设施层 (Infrastructure)

**文件夹**: [infrastructure-layer/](./infrastructure-layer/)

**职责**: 提供持久化、日志等横切关注点，不包含业务逻辑。

**模块列表**:

| 文件 | 组件 | 职责 |
|------|------|------|
| [AppConfigManager.md](./infrastructure-layer/AppConfigManager.md) | `AppConfigManager` | 管理全局应用配置（已注册项目列表、主题等），存储于 `%LOCALAPPDATA%` |
| [ProjectConfigManager.md](./infrastructure-layer/ProjectConfigManager.md) | `ProjectConfigManager` | 管理单个项目的 `projectConfig.json` 读写 |
| [Logger.md](./infrastructure-layer/Logger.md) | `Logger` | 统一日志管理（Serilog），向 Console、File、LogWindow 三处输出 |

**层间约定**: 使用 `System.Text.Json` 进行 JSON 序列化，读写均为异步操作。

---

## Luban 源适配层 (LubanAdapter)

**文件夹**: [luban-adapter-layer/](./luban-adapter-layer/)

**职责**: 作为 GUI 与 `lubanSrc\` 源码之间的**唯一隔离边界**。GUI 所需的 Luban 内部数据（Schema 定义、配置格式、类型系统）全部通过此层接口获取，上层不直接引用任何 `lubanSrc\` 中的类型。

**存在意义**:
- Luban 源码内部 API 随版本迭代可能变化；有此层后，适配改动只需在一处修改。
- 接口语义以 GUI 需求为中心，屏蔽 Luban 内部复杂度。
- 可通过 Mock 此层接口，在不依赖 Luban 源码的情况下独立测试上层逻辑。

**模块列表**:

| 文件 | 组件 | 职责 |
|------|------|------|
| [LubanSchemaReader.md](./luban-adapter-layer/LubanSchemaReader.md) | `LubanSchemaReader` | 读取 Schema 定义（表、枚举、结构体），返回 GUI 友好的 DTO |
| [LubanConfAdapter.md](./luban-adapter-layer/LubanConfAdapter.md) | `LubanConfAdapter` | 解析和生成 `luban.conf`，屏蔽 Luban 配置格式细节 |
| [LubanTypeMapper.md](./luban-adapter-layer/LubanTypeMapper.md) | `LubanTypeMapper` | 将 Luban 内置类型映射为 GUI 可显示的类型描述 |

**层间约定**: 此层是唯一允许 `using` 任何 `lubanSrc\` 命名空间的层，上方各层严禁直接引用。
