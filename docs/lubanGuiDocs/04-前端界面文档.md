## 整体布局概览

Luban-GUI 主窗口采用 **菜单栏 + 项目工具栏 + 左右分栏** 布局：左侧为表格列表（含快捷操作），右侧为选中表格的内容预览；日志输出独立为浮动二级窗口，不占用主界面空间。

## 主窗口

主窗口由五个区域构成：**菜单栏**、**项目切换栏**、**快捷操作工具栏**、**主内容区**（左右分栏）、**状态栏**。

---

### 菜单栏

位于窗口顶部，采用**悬停/点击展开下拉菜单**的形式。菜单栏右侧显示当前操作状态指示器。

#### 菜单结构

**文件(F)**

| 菜单项 | 快捷键 | 说明 |
|--------|--------|------|
| 新建项目... | `Ctrl+N` | 创建新项目（选择工作区路径） |
| 打开项目... | `Ctrl+O` | 打开已有项目文件夹 |
| ─────────── | | |
| 退出 | `Alt+F4` | 退出应用程序 |

**操作(A)**

| 菜单项 | 快捷键 | 说明 |
|--------|--------|------|
| ▶ 全量导表 | `F5` | 验证配置后启动 Luban 进程，执行完整导表 |
| ✅ 校验配置 | `F6` | 仅检查导出配置是否合法，不执行导表 |
| ─────────── | | |
| ⏹ 取消 | `Esc` | 中止正在进行的导表（仅导表中可用） |
| ─────────── | | |
| ⚙ 导出配置... | `Ctrl+,` | 打开导出配置窗口（数据/代码类型和输出路径） |

**视图(V)**

| 菜单项 | 快捷键 | 说明 |
|--------|--------|------|
| 📋 日志窗口 | `Ctrl+L` | 打开/显示日志浮动窗口 |
| ─────────── | | |
| 刷新表格列表 | `F5` | 重新扫描当前项目工作区下的表格文件 |

**帮助(H)**

| 菜单项 | 说明 |
|--------|------|
| 关于 Luban-GUI | 打开关于窗口 |
| 查看文档 | 打开在线文档 |
| 在 GitHub 上查看 | 打开项目仓库页面 |

#### 状态指示器（菜单栏右侧）

| 样式 | 含义 |
|------|------|
| `● 就绪` — 灰色 | 等待操作 |
| `⟳ 导表中… 3s` — 蓝色旋转图标 | 正在执行，实时显示已耗时秒数 |
| `✓ 成功` — 绿色 | 导表完成，3 秒后恢复为就绪 |
| `✗ 失败` — 红色 | 导表失败，常驻直至用户再次操作 |

---

### 项目切换栏

位于菜单栏下方，用于在多个项目之间切换。

```
┌──────────────────────────────────────────────────────────────────────┐
│  项目: [MyGame ▾]   [+ 新建项目]  [📂 打开项目文件夹]               │
└──────────────────────────────────────────────────────────────────────┘
```

**功能说明**:

- **项目下拉框**：列出所有已注册项目，点击切换。切换后表格列表和内容区同步刷新
- **[+ 新建项目]**：弹出对话框，输入项目名称并选择工作区路径，GUI 自动初始化工作区文件结构
- **[📂 打开项目文件夹]**：在系统文件管理器中打开当前项目的工作区目录，方便用户直接拖入 Excel 文件

---

### 快捷操作工具栏

位于项目切换栏下方，提供表格管理的快捷入口。

```
┌──────────────────────────────────────────────────────────────────────┐
│  [+ 新建表格]  [+ 新建枚举]  [+ 新建Bean]  [📥 导入文件]            │
└──────────────────────────────────────────────────────────────────────┘
```

| 按钮 | 行为 |
|------|------|
| **+ 新建表格** | 弹出对话框，输入表名、字段列表（字段名 + 类型），GUI 自动在 `Datas/` 下生成符合 Luban 规范的 xlsx，并写入 `__tables__.xlsx` |
| **+ 新建枚举** | 弹出对话框，输入枚举名和枚举值，GUI 自动写入 `__enums__.xlsx` |
| **+ 新建Bean** | 弹出对话框，输入 Bean 名和字段列表，GUI 自动写入 `__beans__.xlsx` |
| **📥 导入文件** | 打开文件选择对话框，选择已有 xlsx 文件导入当前项目（注册到 `__tables__.xlsx`） |

> **拖入方式**：用户也可以点击「打开项目文件夹」后，直接将 xlsx 文件拖入 `Datas/` 目录，然后点击「刷新表格列表」(`F5`) 以识别新文件。此时 GUI 会弹出提示，询问是否将拖入的文件注册到 `__tables__.xlsx`。

### 主内容区

主内容区水平分为左右两个面板，中间设有可拖动分隔条。

---

#### 左侧：表格列表面板

展示当前项目中所有已注册的 Luban 表格。

```
┌──────────────────────┐
│ 📋 表格列表           │
│──────────────────────│
│ 🔍 [过滤...________] │
│                      │
│ ▾ 全部 (5)           │
│   ├ Item_Config   ✓  │
│   ├ Skill_Config  ✓  │
│   ├ Drop_Table    ✓  │
│   ├ Level_Config  ✓  │
│   └ NPC_Config    ✗  │
│                      │
└──────────────────────┘
```

**功能说明**：

- **过滤框**：实时按名称筛选表格
- **表格条目**：每行显示表格名称，右侧图标表示上次导表状态：
  - `✓` 绿色 — 本次已成功导出
  - `✗` 红色 — 上次导出失败
  - `—` 灰色 — 尚未导出
- **单击**：在右侧面板中预览该表格内容
- **右键菜单**：
  - 单独导出此表
  - 在文件管理器中显示源文件
  - 在系统程序中打开源文件
  - 复制表格名称
  - 从项目中移除（仅移除注册信息，不删除文件）

> 当项目尚未创建/打开时，列表显示空白提示："请先在「文件」中新建或打开一个项目"。

---

#### 右侧：表格内容面板

以表格形式预览左侧选中表格的数据内容。

```
┌──────────────────────────────────────────────────────────────────────┐
│ 📄 Item_Config                                    [↗ 在编辑器中打开] │
│──────────────────────────────────────────────────────────────────────│
│  ┌────┬──────────┬─────────┬──────────────────────────────────────┐  │
│  │ id │ name     │  value  │ desc                                 │  │
│  ├────┼──────────┼─────────┼──────────────────────────────────────┤  │
│  │  1 │ item_001 │     100 │ 基础道具，可在商店购买               │  │
│  │  2 │ item_002 │     200 │ 进阶道具，需合成获得                 │  │
│  │  3 │ item_003 │     300 │ 高级道具，副本掉落                   │  │
│  │  4 │ item_004 │     400 │ 稀有道具，活动专属                   │  │
│  │  … │ …        │       … │ …                                    │  │
│  └────┴──────────┴─────────┴──────────────────────────────────────┘  │
│  共 128 行                                                           │
└──────────────────────────────────────────────────────────────────────┘
```

**功能说明**：

- **表头排序**：点击列标题可按该列升/降序排列
- **行数统计**：底部显示总行数
- **虚拟化渲染**：支持大量数据行不卡顿
- **在编辑器中打开**：点击右上角按钮，用系统默认程序打开源 Excel 文件
- **双击列标题**：用系统默认程序打开源 Excel 文件，并尝试定位到对应列（字段）的位置（在 Windows + Excel 环境下通过 COM 自动化实现；其他环境退化为仅打开文件）
- **无选中时**：面板显示占位提示 "← 请在左侧列表中选择一张表格"
- **读取失败时**：面板显示错误信息，说明原因（如"文件不存在"、"格式错误"等）

## 二级窗口

### 导出配置窗口

通过「操作 > 导出配置」(`Ctrl+,`) 打开的模态对话框，**仅需配置导出类型和输出路径**，无需接触 Luban CLI 参数细节。

```
┌──────────────────── 导出配置 ──────────────────────────────────────┐
│                                                                    │
│  项目: MyGame                                                      │
│                                                                    │
│  ▾ 数据导出 ─────────────────────────────────────────────────     │
│    导出类型:    [bin               ▾]                              │
│    输出路径:    [__________________________________] [📂 浏览]     │
│                                                                    │
│  ▾ 代码导出 ─────────────────────────────────────────────────     │
│    ☑ 生成代码                                                      │
│    代码类型:    [cs-bin            ▾]                              │
│    输出路径:    [__________________________________] [📂 浏览]     │
│    顶层命名空间: [cfg_________________________________]            │
│                                                                    │
│  ▾ 导出目标 ─────────────────────────────────────────────────     │
│    目标:        [all               ▾]                              │
│                  说明: all = 导出 client + server 两端数据         │
│                                                                    │
│                                       [取消]  [保存并关闭]         │
└────────────────────────────────────────────────────────────────────┘
```

#### 数据导出

| 字段 | 说明 | 控件 |
|------|------|------|
| 导出类型 | 数据序列化格式，如 `bin`、`json`、`lua` 等 | 下拉选择 |
| 输出路径 | 导出数据文件的目标目录 | 路径输入框 + 浏览 |

#### 代码导出

| 字段 | 说明 | 控件 |
|------|------|------|
| 生成代码 | 是否同时生成代码文件 | 复选框（不勾选时禁用以下字段） |
| 代码类型 | 代码目标，如 `cs-bin`、`typescript-json`、`java-bin` 等 | 下拉选择 |
| 输出路径 | 生成代码文件的目标目录 | 路径输入框 + 浏览 |
| 顶层命名空间 | 生成代码的顶层模块名，如 `cfg` | 文本框 |

#### 导出目标

| 字段 | 说明 | 控件 |
|------|------|------|
| 目标 | 对应 `luban.conf` 中定义的 `targets`，如 `all`、`client`、`server` | 下拉选择（从当前项目 `luban.conf` 动态读取） |

> 所有 Luban CLI 的底层参数（`--conf`、`--target`、`--codeTarget`、`--dataTarget`、`--xargs outputCodeDir=...` 等）均由 GUI 根据上述配置自动组装，**用户不需要了解也不需要填写任何命令行参数**。

### 日志窗口

通过「视图 > 日志窗口」(`Ctrl+L`) 打开的独立浮动窗口，实时展示 Luban 进程输出。  
窗口关闭后日志内容保留，重新打开时仍可查看本次会话的历史日志。

```
┌──────────────────── 日志 ───────────────────────────────── [─][□][×]┐
│                                                          [🗑 清空]   │
│  [INFO]    15:00:01  就绪，等待导表...                               │
│  [INFO]    15:00:05  验证配置...                                     │
│  [INFO]    15:00:05  启动 Luban 进程...                              │
│  [OUT]               Processing Excel files...                      │
│  [OUT]               Sheet1: 100 rows                               │
│  [WARN]              Missing optional field at Sheet2!B3            │
│  [SUCCESS] 15:00:07  导表完成 (耗时 2.34s)                           │
│                                                                     │
│                                              [↓ 跳到最新]            │
└─────────────────────────────────────────────────────────────────────┘
```

#### 日志级别与颜色

| 前缀 | 含义 | 颜色 |
|------|------|------|
| `[INFO]` | 一般流程信息（GUI 自身产生） | 灰色 |
| `[OUT]` | Luban stdout 原始输出 | 白色/默认色 |
| `[WARN]` | 警告信息（来自 Luban stderr 或 GUI 判断） | 黄色 |
| `[ERROR]` | 错误信息 | 红色 |
| `[SUCCESS]` | 导表成功（GUI 在进程退出码为 0 时追加） | 绿色 |

#### 窗口功能

- **自动弹出**：每次执行"全量导表"或"校验配置"时，若日志窗口未打开则自动打开
- **自动滚动**：默认跟随最新日志；用户手动向上滚动时暂停，出现"↓ 跳到最新"按钮
- **清空**：点击🗑按钮清空日志；每次新导表启动前自动清空
- **全选 + 复制**：支持 `Ctrl+A` 全选、`Ctrl+C` 复制
- **虚拟化渲染**：支持 10,000+ 条日志不卡顿

## 交互行为规范

### 菜单项状态

| 状态 | 全量导表 | 校验配置 | 取消 |
|------|---------|---------|------|
| 初始（未打开项目） | 置灰 | 置灰 | 置灰 |
| 就绪（项目已打开，导出配置已填） | 可用 | 可用 | 置灰 |
| 导表进行中 | 置灰 | 置灰 | 可用 |
| 导表完成 | 可用 | 可用 | 置灰 |

### 新建表格对话框

```
┌──────────── 新建表格 ────────────┐
│  表名: [ItemConfig______________] │
│                                   │
│  字段列表:                        │
│  ┌──────────┬──────────────────┐  │
│  │ 字段名   │ 类型             │  │
│  ├──────────┼──────────────────┤  │
│  │ id       │ int              │  │
│  │ name     │ string           │  │
│  │ value    │ int              │  │
│  └──────────┴──────────────────┘  │
│  [+ 添加字段]                     │
│                                   │
│          [取消]   [创建]          │
└───────────────────────────────────┘
```

- 字段名不能为空，类型从 Luban 内置类型中选择（`int`、`string`、`bool`、`float` 等）
- 点击「创建」后，GUI 在 `Datas/` 下生成对应 xlsx，并更新 `__tables__.xlsx`

### 路径输入框（导出配置窗口内）

- 支持直接拖拽文件夹到输入框
- 路径不存在时，输入框边框高亮为红色，Tooltip 显示具体错误
- 路径有效时，输入框边框为正常色

### 双击列标题（表格内容面板）

1. 获取该列对应的字段名和 Excel 文件路径
2. 调用 `FileOpenService.OpenFileAtCell()`
3. **Windows + Excel**：通过 COM 自动化打开文件并定位到对应列
4. **其他环境**：退化为直接打开 Excel 文件（使用系统默认程序）

### 校验反馈

点击「操作 > 校验配置」后：
- **通过**：菜单栏状态区显示绿色 `✓ 配置有效`，3 秒后恢复；日志窗口自动弹出并追加结果
- **失败**：菜单栏状态区显示红色 `✗ 配置错误`；日志窗口自动弹出并追加具体错误列表：
  ```
  [ERROR] 校验失败，发现以下问题：
  [ERROR]   • 数据输出路径为空
  [ERROR]   • 代码输出路径不存在: D:\NoSuch\Scripts
  ```

### 首次启动引导

应用首次启动时（无已注册项目），弹出提示，引导用户新建或打开一个项目。

## 视觉规范

### 主题

- v1.0 仅提供**浅色主题**（Avalonia 默认）
- v1.1 计划引入暗色主题切换

### 颜色语义

| 用途 | 颜色 |
|------|------|
| 成功、正常 | `#4CAF50`（绿色）|
| 错误 | `#F44336`（红色）|
| 警告 | `#FF9800`（橙黄色）|
| 进行中 | `#2196F3`（蓝色）|
| 禁用/次要 | `#9E9E9E`（灰色）|

### 布局尺寸

- **最小窗口尺寸**：宽 900px × 高 600px
