## 整体布局概览

Luban-GUI 主窗口采用 **菜单栏 + 左右分栏** 布局：左侧为表格列表，右侧为选中表格的内容预览；日志输出独立为浮动二级窗口，不占用主界面空间。

```

│  TITLE BAR: Luban-GUI                               [─] [□] [×]     │

├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌──────────────────┬──────────────────────────────────────────────┐ │
│  │ 📋 表格列表       │ 📄 Item_Config                               │ │
│  │──────────────────│──────────────────────────────────────────────│ │
│  │ 🔍 [过滤...___]  │                                              │ │
│  │                  │  ┌────┬──────────┬─────────┬──────────────┐  │ │
│  │ ▾ 全部 (5)       │  │ id │ name     │  value  │ desc         │  │ │
```

## 主窗口

主窗口由三个区域构成：**菜单栏**、**主内容区**（左右分栏）、**状态栏**。  
左右两侧面板之间设有可拖动的分隔线，用户可自由调整各列宽度。

---

### 菜单栏

位于窗口顶部，参考 VS Code 设计，采用**悬停/点击展开下拉菜单**的形式。鼠标悬停在菜单项上即展开对应下拉面板，点击菜单项外区域关闭。菜单栏右侧显示当前操作状态指示器。

#### 菜单结构

**文件(F)**

| 菜单项 | 快捷键 | 说明 |
|--------|--------|------|
| 新建配置... | `Ctrl+N` | 创建新的导出配置 |
| 打开配置... | `Ctrl+O` | 从文件加载已保存的配置 |
| 保存配置 | `Ctrl+S` | 保存当前配置到文件 |
| 另存为... | `Ctrl+Shift+S` | 将当前配置保存到新文件 |
| ─────────── | | |
| 退出 | `Alt+F4` | 退出应用程序 |

**操作(A)**

| 菜单项 | 快捷键 | 说明 |
|--------|--------|------|
| ▶ 全量导表 | `F5` | 验证配置后启动 Luban 进程，执行完整导表 |
| ✅ 校验配置 | `F6` | 仅检查配置项是否合法，不执行导表 |
| ─────────── | | |
| ⏹ 取消 | `Esc` | 中止正在进行的导表（仅导表中可用） |
| ─────────── | | |
| ⚙ 导出设置... | `Ctrl+,` | 打开导出配置窗口（Luban 路径、参数等） |

**视图(V)**

| 菜单项 | 快捷键 | 说明 |
|--------|--------|------|
| 📋 日志窗口 | `Ctrl+L` | 打开/显示日志浮动窗口 |
| ─────────── | | |
| 刷新表格列表 | `F5` | 重新扫描配置文件中的表格 |

**帮助(H)**

| 菜单项 | 说明 |
|--------|------|
| 关于 Luban-GUI | 打开关于窗口 |
| 查看文档 | 打开在线文档 |
| 在 GitHub 上查看 | 打开项目仓库页面 |

#### 状态指示器（菜单栏右侧）

| 样式 | 含义 |
|------|------|
| `● 就绪` — 灰色 | 等待操作 |
| `⟳ 导表中… 3s` — 蓝色旋转图标 | 正在执行，实时显示已耗时秒数 |
| `✓ 成功` — 绿色 | 导表完成，3 秒后恢复为就绪 |
| `✗ 失败` — 红色 | 导表失败，常驻直至用户再次操作 |

### 主内容区

主内容区水平分为左右两个面板，中间设有可拖动分隔条。

---

#### 左侧：表格列表面板

展示当前导出配置中所有已识别的 Luban 表格，支持快速定位与筛选。

```
┌──────────────────────┐
│ 📋 表格列表           │
│──────────────────────│
│ 🔍 [过滤...________] │
│                      │
│ ▾ 全部 (5)           │
│   ├ Item_Config   ✓  │
│   ├ Skill_Config  ✓  │
│   ├ Drop_Table    ✓  │
│   ├ Level_Config  ✓  │
│   └ NPC_Config    ✗  │
│                      │
└──────────────────────┘
```

**功能说明**：

- **过滤框**：实时按名称筛选表格，输入时列表即时更新
- **表格条目**：每行显示表格名称，右侧图标表示上次导表状态：
  - `✓` 绿色 — 本次已成功导出
  - `✗` 红色 — 上次导出失败
  - `—` 灰色 — 尚未导出
- **单击**：在右侧面板中预览该表格内容
- **右键菜单**：
  - 单独导出此表
  - 在文件管理器中显示源文件
  - 复制表格名称

> 当导出配置尚未设置时，列表显示空白提示："请先在「操作 > 导出设置」中配置 Luban 路径和配置文件"。

---

#### 右侧：表格内容面板

以表格形式预览左侧选中表格的数据内容。

```
┌──────────────────────────────────────────────────────────────────────┐
│ 📄 Item_Config                                    [↗ 在编辑器中打开] │
│──────────────────────────────────────────────────────────────────────│
│  ┌────┬──────────┬─────────┬──────────────────────────────────────┐  │
│  │ id │ name     │  value  │ desc                                 │  │
│  ├────┼──────────┼─────────┼──────────────────────────────────────┤  │
│  │  1 │ item_001 │     100 │ 基础道具，可在商店购买               │  │
│  │  2 │ item_002 │     200 │ 进阶道具，需合成获得                 │  │
│  │  3 │ item_003 │     300 │ 高级道具，副本掉落                   │  │
│  │  4 │ item_004 │     400 │ 稀有道具，活动专属                   │  │
│  │  … │ …        │       … │ …                                    │  │
│  └────┴──────────┴─────────┴──────────────────────────────────────┘  │
│  共 128 行                                                           │
└──────────────────────────────────────────────────────────────────────┘
```

**功能说明**：

- **表头排序**：点击列标题可按该列升/降序排列
- **行数统计**：底部显示总行数
- **虚拟化渲染**：支持大量数据行不卡顿
- **在编辑器中打开**：点击右上角按钮，用系统默认程序打开源数据文件（Excel / CSV）
- **无选中时**：面板显示占位提示 "← 请在左侧列表中选择一张表格"
- **读取失败时**：面板显示错误信息，说明原因（如"配置文件不存在"、"解析错误"等）

## 二级窗口

### 导出配置窗口

通过「操作 > 导出设置」(`Ctrl+,`) 打开的模态对话框，集中管理所有传递给 Luban CLI 的参数。

```
┌──────────────────── 导出设置 ──────────────────────────────────────┐
│                                                                    │
│  配置名称: [DefaultGame________________________________]           │
│                                                                    │
│  ▾ 必填参数 ─────────────────────────────────────────────────     │
│    Luban 路径:  [__________________________________] [📂 浏览]     │
│    配置文件:    [__________________________________] [📂 浏览]     │
│    目标:        [__________________________________]               │
│                                                                    │
│  ▾ 常用可选 ─────────────────────────────────────────────────     │
│    代码目标:    [cs-bin ×] [typescript-json ×]  [+ 添加]           │
│    数据目标:    [bin ×]                         [+ 添加]           │
│    扩展参数:    [outputCodeDir=D:\Gen ×]        [+ 添加]           │
│                                                                    │
│  ▸ 高级参数 ─────────────────────────────────────────────────     │
│                                                                    │
│                                       [取消]  [保存并关闭]         │
└────────────────────────────────────────────────────────────────────┘
```

#### 必填参数

| 字段 | CLI 参数 | 控件 |
|------|---------|------|
| Luban 路径 | （可执行文件路径） | 路径输入框 + 浏览 |
| 配置文件 | `--conf` | 路径输入框 + 浏览 |
| 目标 | `--target` | 文本框 |

> 三个字段均为必填，任何一项为空时主窗口「全量导表」和「校验配置」菜单项置灰。

#### 常用可选参数

| 字段 | CLI 参数 | 控件 |
|------|---------|------|
| 代码目标 | `--codeTarget` | 多值标签输入 |
| 数据目标 | `--dataTarget` | 多值标签输入 |
| 扩展参数 | `--xargs` | 多值标签输入（key=value）|

#### 高级参数（默认折叠）

| 字段标签 | CLI 参数 | 控件类型 |
|---------|---------|---------|
| Schema 收集器 | `--schemaCollector` | 文本框 |
| Pipeline | `--pipeline` | 文本框 |
| 只输出指定表 | `--outputTable` | 多值标签输入 |
| 包含标签 | `--includeTag` | 多值标签输入 |
| 排除标签 | `--excludeTag` | 多值标签输入 |
| 字段变体 | `--variant` | 多值标签输入（key=value）|
| 时区 | `--timeZone` | 文本框 |
| 自定义模板目录 | `--customTemplateDir` | 多值路径输入 + 浏览 |
| 验证失败作为错误 | `--validationFailAsError` | 复选框 |
| 监听目录 | `--watchDir` | 多值路径输入 + 浏览 |
| 强制加载表数据 | `--forceLoadTableDatas` | 复选框 |
| Verbose 输出 | `--verbose` | 复选框 |
| NLog 配置文件 | `--logConfig` | 文本框 + 浏览 |

### 日志窗口

通过「视图 > 日志窗口」(`Ctrl+L`) 打开的独立浮动窗口，实时展示 Luban 进程输出。  
窗口关闭后日志内容保留，重新打开时仍可查看本次会话的历史日志。

```
┌──────────────────── 日志 ───────────────────────────────── [─][□][×]┐
│                                                          [🗑 清空]   │
│  [INFO]    15:00:01  就绪，等待导表...                               │
│  [INFO]    15:00:05  验证配置...                                     │
│  [INFO]    15:00:05  启动 Luban 进程...                              │
│  [OUT]               Processing Excel files...                      │
│  [OUT]               Sheet1: 100 rows                               │
│  [WARN]              Missing optional field at Sheet2!B3            │
│  [SUCCESS] 15:00:07  导表完成 (耗时 2.34s)                           │
│                                                                     │
│                                              [↓ 跳到最新]            │
└─────────────────────────────────────────────────────────────────────┘
```

#### 日志级别与颜色

| 前缀 | 含义 | 颜色 |
|------|------|------|
| `[INFO]` | 一般流程信息（GUI 自身产生） | 灰色 |
| `[OUT]` | Luban stdout 原始输出 | 白色/默认色 |
| `[WARN]` | 警告信息（来自 Luban stderr 或 GUI 判断） | 黄色 |
| `[ERROR]` | 错误信息 | 红色 |
| `[SUCCESS]` | 导表成功（GUI 在进程退出码为 0 时追加） | 绿色 |

#### 窗口功能

- **自动弹出**：每次执行"全量导表"或"校验配置"时，若日志窗口未打开则自动打开
- **自动滚动**：默认跟随最新日志；用户手动向上滚动时暂停，出现"↓ 跳到最新"按钮
- **清空**：点击🗑按钮清空日志；每次新导表启动前自动清空
- **全选 + 复制**：支持 `Ctrl+A` 全选、`Ctrl+C` 复制
- **虚拟化渲染**：支持 10,000+ 条日志不卡顿

## 交互行为规范

### 菜单项状态

| 状态 | 全量导表 | 校验配置 | 取消 |
|------|---------|---------|------|
| 初始（必填项为空） | 置灰 | 置灰 | 置灰 |
| 就绪（必填项已填） | 可用 | 可用 | 置灰 |
| 导表进行中 | 置灰 | 置灰 | 可用 |
| 导表完成 | 可用 | 可用 | 置灰 |

### 路径输入框（导出配置窗口内）

- 支持直接拖拽文件/文件夹到输入框
- 路径不存在时，输入框边框高亮为红色，Tooltip 显示具体错误
- 路径有效时，输入框边框为正常色

### 多值标签输入（Tag Input）

用于 `--codeTarget`、`--dataTarget`、`--xargs` 等多值参数：

```
[cs-bin ×]  [typescript-json ×]   [+ 添加]
```

- 点击 `+ 添加` 弹出内联输入框，回车确认，`Escape` 取消
- 点击标签上的 `×` 删除该项
- `--xargs` 和 `--variant` 类型在失焦时验证 `key=value` 格式，不符合时显示红色边框

### 校验反馈

点击「操作 > 校验配置」后：
- **通过**：菜单栏状态区显示绿色 `✓ 配置有效`，3 秒后恢复；日志窗口自动弹出并追加结果
- **失败**：菜单栏状态区显示红色 `✗ 配置错误`；日志窗口自动弹出并追加具体错误列表：
  ```
  [ERROR] 校验失败，发现以下问题：
  [ERROR]   • Luban 路径不存在: C:\NoSuch\luban.exe
  [ERROR]   • xargs 格式错误: "badformat"（应为 key=value）
  ```

### 首次启动引导

应用首次启动时（无保存配置），弹出 **导出配置窗口** 并提示用户填写必填参数，填写完成后自动关闭并加载表格列表。

## 视觉规范

### 主题

- v1.0 仅提供**浅色主题**（Avalonia 默认）
- v1.1 计划引入暗色主题切换

### 颜色语义

| 用途 | 颜色 |
|------|------|
| 成功、正常 | `#4CAF50`（绿色）|
| 错误 | `#F44336`（红色）|
| 警告 | `#FF9800`（橙黄色）|
| 进行中 | `#2196F3`（蓝色）|
| 禁用/次要 | `#9E9E9E`（灰色）|

### 布局尺寸

- **最小窗口尺寸**：宽 800px × 高 600px
- **默认窗口尺寸**：宽 1100px × 高 750px
- **左侧表格列表面板** 默认宽度：240px，最小 160px
- **菜单栏**：高度 30px（与一般桌面应用风格对齐）
- **日志窗口** 默认尺寸：宽 700px × 高 400px，可调整大小

## 后续版本规划

### v1.1：多配置支持

在侧边栏上方新增**配置切换下拉框**，支持管理多套导出配置（对应不同项目或目标）。

```
┌──────────────────────────────────────────────────────────────────────┐
│  文件(F)   操作(A)   视图(V)   帮助(H)                               │
├──────────────────────────────────────────────────────────────────────┤
│  当前配置: [DefaultGame ▾]   [+ 新建]  [复制]  [删除]               │
├──────────────────┬───────────────────────────────────────────────────┤
│ 📋 表格列表       │ 📄 表格内容                                      │
│  ...             │  ...                                              │
└──────────────────┴───────────────────────────────────────────────────┘
```

- 配置列表支持新建、复制、删除、重命名
- 支持配置文件导入/导出（JSON 格式）
- 切换配置时，表格列表和内容区域同步刷新

### v1.2：日志增强

- 日志窗口顶部增加过滤工具栏（按级别筛选、关键字搜索）
- 支持将日志导出为文本文件
- 显示导表历史记录（可点击查看历史日志）
