# Luban 元数据表格式规范

> 本文档基于 `LubanGui/Tmplate/Datas/` 下的官方模板文件 `__tables__.xlsx`、`__enums__.xlsx`、`__beans__.xlsx` 实际结构分析整理，供 `ExcelWriter` 等代码逻辑参考。

---

## 一、`__tables__.xlsx`

### Sheet 结构

| 列 | A | B | C | D | E | F | G | H | I | J | K |
|----|---|---|---|---|---|---|---|---|---|---|---|
| **Row 1** | `##var` | `full_name` | `value_type` | `read_schema_from_file` | `input` | `index` | `mode` | `group` | `comment` | `tags` | `output` |
| **Row 2** | `##` | 全名(包含模块和名字) | 记录类名 | 从excel读取定义 | 文件列表 | 表id字段 | 模式 | 分组 | 注释 | *(空)* | 输出文件名 |
| **Row 3** | `##` | *(详细说明...)* | … | … | … | … | … | … | … | … | … |
| 数据行 | *(空)* | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

### 字段列顺序（B 列起）

| 列号 | 字段名 | 说明 |
|------|--------|------|
| B (2) | `full_name` | 全名，包含模块和名字（如 `item.Item`） |
| C (3) | `value_type` | 记录类名（Bean 名） |
| D (4) | `read_schema_from_file` | 是否从 Excel 读取 Schema（`true`/`false`） |
| E (5) | `input` | 输入文件，多个以逗号分隔 |
| F (6) | `index` | 主键字段名，空则取 `value_type` 第一个字段 |
| G (7) | `mode` | 表模式：`one`/`map`/`list`，空默认为 `map` |
| H (8) | `group` | 分组：`c`/`s`/`e`，空为全分组 |
| I (9) | `comment` | 注释 |
| J (10) | `tags` | 标签 |
| K (11) | `output` | 输出文件名，默认 `<module>_<name>.<suffix>` |

### 注意事项

- `##var` 是第一行，无独立 `##` meta 行前置
- Row 2、Row 3 均为 `##` 注释行（人类可读说明，不参与解析）
- 字段列无 `##type` 行

## 二、`__enums__.xlsx`

### Sheet 结构（含合并单元格）

| 列 | A | B | C | D | E | F | G | H~L（合并） |
|----|---|---|---|---|---|---|---|-------------|
| **Row 1** | `##var` | `full_name` | `flags` | `unique` | `group` | `comment` | `tags` | `*items`（H1:L1 合并） |
| **Row 2** | `##var` | *(空)* | *(空)* | *(空)* | *(空)* | *(空)* | *(空)* | `name` \| `alias` \| `value` \| `comment` \| `tags` |
| **Row 3** | `##` | 全名(包含模块和名字) | 是否为位标记枚举 | 枚举项是否唯一 | *(空)* | *(空)* | *(空)* | 枚举名 \| 别名 \| 值 \| 注释 \| *(空)* |
| 数据行 | *(空)* | ... | ... | ... | ... | ... | ... | ... |

### 顶层字段（B~G 列）

| 列号 | 字段名 | 说明 |
|------|--------|------|
| B (2) | `full_name` | 枚举全名，包含模块 |
| C (3) | `flags` | 是否为位标记枚举（`true`/`false`） |
| D (4) | `unique` | 枚举项值是否唯一（`true`/`false`） |
| E (5) | `group` | 分组 |
| F (6) | `comment` | 注释 |
| G (7) | `tags` | 标签 |

### `*items` 容器子字段（H~L 列，H1:L1 合并）

| 列号 | 子字段名 | 说明 |
|------|----------|------|
| H (8)  | `name`    | 枚举项名称 |
| I (9)  | `alias`   | 别名 |
| J (10) | `value`   | 枚举值 |
| K (11) | `comment` | 注释 |
| L (12) | `tags`    | 标签 |

### 注意事项

- 有**两行 `##var`**：第一行定义顶层字段，第二行定义 `*items` 子字段
- `*items` 在第一行 `##var` 中占据 H1:L1 **合并单元格**
- `*items` 使用多行模式：一条枚举记录的每个枚举项占据独立的数据行（`full_name` 等顶层字段只在第一行填写）

- `*fields` 使用多行模式：一条 Bean 记录的每个字段占据独立数据行

## 四、`ExcelWriter` 代码对应关系

| 方法 | 对应文件 | 关键点 |
|------|----------|--------|
| `CreateTablesMetaXlsx` | `__tables__.xlsx` | 单行 `##var`，无合并单元格 |
| `CreateEnumsMetaXlsx`  | `__enums__.xlsx`  | 双行 `##var`，H1:L1 合并 |
| `CreateBeansMetaXlsx`  | `__beans__.xlsx`  | 双行 `##var`，J1:P1 合并 |
| `AppendTableEntry`     | `__tables__.xlsx` | 列映射须与 `s_tableFields` 顺序一致 |

## 五、ExcelWriter 字段顺序速查

### `__tables__.xlsx` 字段顺序

```
B=full_name  C=value_type  D=read_schema_from_file  E=input
F=index  G=mode  H=group  I=comment  J=tags  K=output
```

### `__enums__.xlsx` 字段顺序

```
顶层: B=full_name  C=flags  D=unique  E=group  F=comment  G=tags
子字段(*items, H~L): name  alias  value  comment  tags
```

### `__beans__.xlsx` 字段顺序

```
顶层: B=full_name  C=parent  D=valueType  E=sep  F=alias  G=comment  H=group  I=tags
子字段(*fields, J~P): name  alias  type  group  comment  tags  variants
```
