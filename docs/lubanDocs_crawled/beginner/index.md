## 新手教程

[## 📄️快速上手 安装](https://www.datable.cn/docs/beginner/quickstart)[## 📄️生成代码和数据 Luban不仅支持导出配置数据，也内置支持生成多种语言的代码，用于运行时加载配置。同时Luban也支持protobuf、flatbuffers、msgpack等流行的消息框架。](https://www.datable.cn/docs/beginner/generatecodeanddata)[## 📄️集成到项目 安装 dotnet SDK](https://www.datable.cn/docs/beginner/integratetoproject)[## 📄️运行时加载配置 我们已经在Projects下提供了大量示例项目。](https://www.datable.cn/docs/beginner/loadinruntime)[## 📄️使用自定义类型 实践中经常遇到定义枚举和自定义结构的情况，Luban完美支持这个特性。](https://www.datable.cn/docs/beginner/usecustomtype)[## 📄️使用容器类型 容器是也是最常用的类型。Luban支持丰富的容器数据格式，可以按照需求选择最适合的类型和填写。](https://www.datable.cn/docs/beginner/usecollection)[## 📄️使用列限定与紧凑格式 像结构与容器都是包含多个元素的数据类型，Luban提供了多种方式（不限于以下这几种）读取这种复合数据类型：](https://www.datable.cn/docs/beginner/streamandcolumnformat)[## 📄️使用数据校验器 实践中经常遇到某个字段需要校验合法性。例如item_id字段必须是有一个有效的item表的id。Luban支持完善的数据校验。](https://www.datable.cn/docs/beginner/usevalidator)[## 📄️使用多态类型 并不是所有数据结构都是规范一致的，在复杂的GamePlay玩法中，经常出现Buff效果有非常多种类型，每种类型的字段都不一样。](https://www.datable.cn/docs/beginner/usepolymorphismtype)[## 📄️自动导入table v3.0.0版本起支持自动导入table。](https://www.datable.cn/docs/beginner/importtable)
